Features

  • Powerful search functions
  • comes with notification capable android app
  • Designed with search engine optimization in mind
  • Rating system: users can rate agents, agencies and properties
  • Verification email: Emails are verified upon sign up to confirm genuine agents
  • Property approval system: Properties posted by users can only appear on the website after being confirmed by the admin
  • Notification system: Admins are notified via email upon property creation. users are also notified when their properties are rejected or accepted
  • 3 user category system: Agents, Agencies and customers
  • Sorting system: Sort search results based on recency,price and views
  • Analytics system: users can see how many views their properties received, total number of properties, total profile views, bookmarked properties and rating.
  • Property request system: create requests and agents or agencies that have the property will contact you
  • Built with search engine optimization in mind to help your properties appear and rank well on google and other search engines
  • Beautiful website with great user experience designed with modern tools
